// ═══════════════════════════════════════════════════════════════ // CIROX · BACKEND COMPLETO · Google Apps Script // 4 hojas: CREDITO | FINANZAS | CLASES | VENTAS // // INSTALACIÓN: // 1. Copia la URL y pégala en CIROX → botón SYNC // ═══════════════════════════════════════════════════════════════ // ── CONFIGURACIÓN ────────────────────────────────────────────── const SHEETS = { CREDITO: 'CREDITO', FINANZAS: 'FINANZAS', CLASES: 'CLASES', VENTAS: 'VENTAS', }; const HEADERS = { CREDITO: ['ID','TIPO','DESCRIPCION','CATEGORIA','VALOR','FECHA','CUOTAS','FECHA_REGISTRO'], FINANZAS: ['MES_KEY','TIPO_ITEM','ID_ITEM','NOMBRE','VALOR','PAGADO','CATEGORIA','ORDEN','FECHA_REGISTRO'], CLASES: ['ID','DESCRIPCION','VALOR','FECHA','ESTADO','FECHA_REGISTRO'], VENTAS: ['ID','CLIENTE','FECHA','TOTAL_PAGADO','BASE_GRAVABLE','IVA','RETEFUENTE','RETEICA','COMISION','APLICA_RETE','APLICA_ICA','ESTADO','FECHA_REGISTRO'], }; const COL_STYLES = { CREDITO: {header:'#1a1f35', data:['#141824','#1a1f35']}, FINANZAS: {header:'#0d2137', data:['#0f1a2e','#142240']}, CLASES: {header:'#1a0d35', data:['#160e2a','#1e1240']}, VENTAS: {header:'#0d2e1a', data:['#0d2218','#122e20']}, }; // ── CORS HELPER ──────────────────────────────────────────────── function output(data) { return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function doOptions(e) { return output({ ok: true }); } // ═══════════════════════════════════════════════════════════════ // doGet — LEER TODOS LOS DATOS // Devuelve: { ok, credito:[], finanzas:[], clases:[], ventas:[] } // ═══════════════════════════════════════════════════════════════ function doGet(e) { try { const ss = SpreadsheetApp.getActiveSpreadsheet(); inicializarHojas(ss); const modulo = e && e.parameter && e.parameter.modulo; const result = { ok: true }; if (!modulo || modulo === 'credito' || modulo === 'all') { result.credito = leerHoja(ss, SHEETS.CREDITO, row => ({ id: row[0], tipo: row[1], desc: row[2], cat: row[3], valor: Number(row[4]) || 0, fecha: formatFecha(row[5]), cuotas: Number(row[6]) || 1, })); } if (!modulo || modulo === 'finanzas' || modulo === 'all') { result.finanzas = leerHoja(ss, SHEETS.FINANZAS, row => ({ mesKey: row[0], tipoItem:row[1], idItem: row[2], nombre: row[3], valor: Number(row[4]) || 0, pagado: row[5] === true || row[5] === 'TRUE' || row[5] === 'true', cat: row[6], orden: Number(row[7]) || 99, })); } if (!modulo || modulo === 'clases' || modulo === 'all') { result.clases = leerHoja(ss, SHEETS.CLASES, row => ({ id: row[0], desc: row[1], valor: Number(row[2]) || 0, fecha: formatFecha(row[3]), estado: row[4], })); } if (!modulo || modulo === 'ventas' || modulo === 'all') { result.ventas = leerHoja(ss, SHEETS.VENTAS, row => ({ id: row[0], desc: row[1], fecha: formatFecha(row[2]), totalPagado: Number(row[3]) || 0, base: Number(row[4]) || 0, iva: Number(row[5]) || 0, retefuente: Number(row[6]) || 0, reteica: Number(row[7]) || 0, comision: Number(row[8]) || 0, aplicaRete: row[9] === true || row[9] === 'TRUE', aplicaIca: row[10] === true || row[10] === 'TRUE', estado: row[11], })); } return output(result); } catch(err) { return output({ ok: false, error: err.message, stack: err.stack }); } } // ═══════════════════════════════════════════════════════════════ // doPost — ESCRIBIR / ELIMINAR DATOS // Body: { modulo, accion, ...datos } // ═══════════════════════════════════════════════════════════════ function doPost(e) { try { // Apps Script recibe text/plain cuando el fetch usa mode:no-cors // También acepta application/json para llamadas directas let body; const raw = e.postData ? e.postData.contents : '{}'; try { body = JSON.parse(raw); } catch(parseErr) { return output({ ok: false, error: 'JSON inválido: ' + raw.slice(0, 100) }); } const modulo = body.modulo || 'credito'; const accion = body.accion || 'nuevo'; const ss = SpreadsheetApp.getActiveSpreadsheet(); inicializarHojas(ss); if (accion === 'eliminar') { return output(eliminarFila(ss, modulo, body.id)); } if (accion === 'sync_completo') { return output(syncCompleto(ss, body)); } // ── INSERTAR según módulo ── switch(modulo) { case 'credito': return output(insertarCredito(ss, body)); case 'finanzas': return output(insertarFinanzas(ss, body)); case 'clases': return output(insertarClase(ss, body)); case 'ventas': return output(insertarVenta(ss, body)); default: return output({ ok: false, error: 'Módulo desconocido: '+modulo }); } } catch(err) { return output({ ok: false, error: err.message }); } } // ── CRÉDITO ──────────────────────────────────────────────────── function insertarCredito(ss, d) { const sheet = ss.getSheetByName(SHEETS.CREDITO); const fila = [ d.id || Date.now().toString(), d.tipo || 'compra', d.desc || '', d.cat || '', Number(d.valor) || 0, d.fecha || '', Number(d.cuotas) || 1, new Date().toISOString().split('T')[0], ]; const row = appendRow(sheet, fila); formatearFila(sheet, row, SHEETS.CREDITO); return { ok: true, accion: 'insertado', modulo: 'credito', fila: row }; } // ── FINANZAS ─────────────────────────────────────────────────── function insertarFinanzas(ss, d) { const sheet = ss.getSheetByName(SHEETS.FINANZAS); const fila = [ d.mesKey || '', d.tipoItem|| 'gasto', d.idItem || Date.now().toString(), d.nombre || '', Number(d.valor) || 0, d.pagado ? 'TRUE' : 'FALSE', d.cat || 'Varios', Number(d.orden) || 99, new Date().toISOString().split('T')[0], ]; const row = appendRow(sheet, fila); formatearFila(sheet, row, SHEETS.FINANZAS); return { ok: true, accion: 'insertado', modulo: 'finanzas', fila: row }; } // ── CLASES ───────────────────────────────────────────────────── function insertarClase(ss, d) { const sheet = ss.getSheetByName(SHEETS.CLASES); const fila = [ d.id || Date.now().toString(), d.desc || '', Number(d.valor) || 0, d.fecha || '', d.estado|| 'pendiente', new Date().toISOString().split('T')[0], ]; const row = appendRow(sheet, fila); formatearFila(sheet, row, SHEETS.CLASES); return { ok: true, accion: 'insertado', modulo: 'clases', fila: row }; } // ── VENTAS ───────────────────────────────────────────────────── function insertarVenta(ss, d) { const sheet = ss.getSheetByName(SHEETS.VENTAS); const fila = [ d.id || Date.now().toString(), d.desc || '', d.fecha || '', Number(d.totalPagado) || 0, Number(d.base) || 0, Number(d.iva) || 0, Number(d.retefuente) || 0, Number(d.reteica) || 0, Number(d.comision) || 0, d.aplicaRete ? 'SI' : 'NO', d.aplicaIca ? 'SI' : 'NO', d.estado || 'cobrado', new Date().toISOString().split('T')[0], ]; const row = appendRow(sheet, fila); formatearFila(sheet, row, SHEETS.VENTAS); return { ok: true, accion: 'insertado', modulo: 'ventas', fila: row }; } // ── SYNC COMPLETO (reemplaza toda una hoja) ──────────────────── function syncCompleto(ss, body) { const resultados = {}; if (body.credito && Array.isArray(body.credito)) { const sh = ss.getSheetByName(SHEETS.CREDITO); limpiarDatos(sh); body.credito.forEach(m => insertarCredito(ss, m)); resultados.credito = body.credito.length; } if (body.finanzas && Array.isArray(body.finanzas)) { const sh = ss.getSheetByName(SHEETS.FINANZAS); limpiarDatos(sh); body.finanzas.forEach(m => insertarFinanzas(ss, m)); resultados.finanzas = body.finanzas.length; } if (body.clases && Array.isArray(body.clases)) { const sh = ss.getSheetByName(SHEETS.CLASES); limpiarDatos(sh); body.clases.forEach(m => insertarClase(ss, m)); resultados.clases = body.clases.length; } if (body.ventas && Array.isArray(body.ventas)) { const sh = ss.getSheetByName(SHEETS.VENTAS); limpiarDatos(sh); body.ventas.forEach(m => insertarVenta(ss, m)); resultados.ventas = body.ventas.length; } return { ok: true, accion: 'sync_completo', resultados }; } // ── ELIMINAR FILA ────────────────────────────────────────────── function eliminarFila(ss, modulo, id) { const nombres = { credito: SHEETS.CREDITO, finanzas: SHEETS.FINANZAS, clases: SHEETS.CLASES, ventas: SHEETS.VENTAS }; const sh = ss.getSheetByName(nombres[modulo]); if (!sh) return { ok: false, error: 'Hoja no encontrada' }; const datos = sh.getDataRange().getValues(); for (let i = 1; i < datos.length; i++) { if (String(datos[i][0]) === String(id)) { sh.deleteRow(i + 1); return { ok: true, accion: 'eliminado', id }; } } return { ok: false, error: 'ID no encontrado: ' + id }; } // ══════════════════════════════════════════════════════════════ // HELPERS // ══════════════════════════════════════════════════════════════ function leerHoja(ss, nombre, mapper) { const sh = ss.getSheetByName(nombre); if (!sh) return []; const rows = sh.getDataRange().getValues(); if (rows.length <= 1) return []; return rows.slice(1) .filter(r => r[0] !== '' && r[0] !== null && r[0] !== undefined) .map(r => { try { return mapper(r); } catch(e) { return null; } }) .filter(Boolean); } function appendRow(sheet, fila) { const lastRow = Math.max(sheet.getLastRow(), 1); const newRow = lastRow + 1; sheet.getRange(newRow, 1, 1, fila.length).setValues([fila]); return newRow; } function formatearFila(sheet, rowNum, sheetName) { const style = COL_STYLES[sheetName]; if (!style) return; const bg = rowNum % 2 === 0 ? style.data[0] : style.data[1]; const numCols = HEADERS[sheetName].length; sheet.getRange(rowNum, 1, 1, numCols) .setBackground(bg) .setFontColor('#e2e8f0') .setFontFamily('Arial') .setFontSize(9) .setVerticalAlignment('middle'); sheet.setRowHeight(rowNum, 22); // Formato moneda en columnas de valor const moneyMap = { CREDITO: [5], FINANZAS: [5], CLASES: [3], VENTAS: [4,5,6,7,8,9], }; const moneyCols = moneyMap[sheetName] || []; moneyCols.forEach(col => { if (col <= numCols) { sheet.getRange(rowNum, col).setNumberFormat('$#,##0'); } }); } function limpiarDatos(sheet) { const last = sheet.getLastRow(); if (last > 1) { sheet.deleteRows(2, last - 1); } } function formatFecha(val) { if (!val) return ''; if (val instanceof Date) { const d = val.getDate().toString().padStart(2,'0'); const m = (val.getMonth()+1).toString().padStart(2,'0'); return `${val.getFullYear()}-${m}-${d}`; } return String(val).split('T')[0]; } // ══════════════════════════════════════════════════════════════ // INICIALIZAR HOJAS (crea si no existen con formato) // ══════════════════════════════════════════════════════════════ function inicializarHojas(ss) { Object.entries(SHEETS).forEach(([key, nombre]) => { let sh = ss.getSheetByName(nombre); if (!sh) { sh = ss.insertSheet(nombre); crearEncabezados(sh, key); } else if (sh.getLastRow() === 0) { crearEncabezados(sh, key); } }); } function crearEncabezados(sheet, sheetKey) { const headers = HEADERS[sheetKey]; const style = COL_STYLES[sheetKey]; sheet.getRange(1, 1, 1, headers.length) .setValues([headers]) .setBackground(style.header) .setFontColor('#ffffff') .setFontFamily('Arial') .setFontSize(9) .setFontWeight('bold') .setHorizontalAlignment('center') .setVerticalAlignment('middle'); sheet.setRowHeight(1, 30); sheet.setFrozenRows(1); // Anchos de columna const widthMap = { CREDITO: [130, 80, 200, 120, 100, 100, 70, 120], FINANZAS: [100, 90, 130, 200, 100, 70, 100, 60, 120], CLASES: [130, 220, 100, 100, 110, 120], VENTAS: [130, 180, 100, 110, 110, 90, 110, 90, 100, 90, 80, 100, 120], }; const widths = widthMap[sheetKey] || []; widths.forEach((w, i) => sheet.setColumnWidth(i+1, w)); } // ══════════════════════════════════════════════════════════════ // MENÚ (opcional — para abrir desde Sheets) // ══════════════════════════════════════════════════════════════ function onOpen() { SpreadsheetApp.getUi() .createMenu('🔴 CIROX') .addItem('📊 Inicializar todas las hojas', 'setupTodasLasHojas') .addItem('🗑️ Limpiar hoja CRÉDITO', 'limpiarCredito') .addItem('📋 Ver resumen', 'verResumen') .addToUi(); } function setupTodasLasHojas() { inicializarHojas(SpreadsheetApp.getActiveSpreadsheet()); SpreadsheetApp.getUi().alert('✅ Hojas creadas: CREDITO | FINANZAS | CLASES | VENTAS'); } function limpiarCredito() { const ui = SpreadsheetApp.getUi(); const r = ui.alert('¿Limpiar todos los datos de CRÉDITO?', ui.ButtonSet.YES_NO); if (r === ui.Button.YES) { limpiarDatos(SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEETS.CREDITO)); ui.alert('✅ Hoja CRÉDITO limpiada.'); } } function verResumen() { const ss = SpreadsheetApp.getActiveSpreadsheet(); inicializarHojas(ss); const countRows = nombre => { const sh = ss.getSheetByName(nombre); return sh ? Math.max(0, sh.getLastRow() - 1) : 0; }; SpreadsheetApp.getUi().alert('📊 Resumen CIROX', '💳 Crédito: ' + countRows(SHEETS.CREDITO) + ' movimientos\n' + '💰 Finanzas: ' + countRows(SHEETS.FINANZAS) + ' ítems\n' + '📚 Clases: ' + countRows(SHEETS.CLASES) + ' clases\n' + '📈 Ventas: ' + countRows(SHEETS.VENTAS) + ' ventas', SpreadsheetApp.getUi().ButtonSet.OK ); }
